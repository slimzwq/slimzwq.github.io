printf

格式字符 说明
d 十进制带符号整数
f 小数输出float/double
e/E 指数输出float/double (大小写表示指数e的大小写)
g/G 以f/e中较短的宽度输出float/double
c 单个字符
s 字符串
p 输出指针地址

整形数据类型拓展

  • x/X 表示16进制无符号
  • o 表示八进制无符号
  • #x/X/o 带前缀输出十六进制/八进制
  • u表示unsigned int
  • hd表示short int(同样用于8/16进制/无符号)
  • ld表示long int
  • lld表示long long int

PS1. short 可能小于int；long可能大于int；long long 可能大于long，具体看不同的硬件。

PS2. char 规定只占1字节，point规定符号处理器位数（比如64位cpu的指针肯定是8B），其他数据类型位数和编译器有关。

数据类型

小数默认double类型，要使用float类型使用1.3f

抽象层级

门级建模

门级建模可以认为是用HDL规定的语言将一个逻辑电路图表示出来。换而言之，就是用Verilog语言中内置的12个基本门级元件，去表述逻辑电路图中元件与元件之间的连接关系。

数据流级建模

到了数据流级建模的阶段，我们去描述电路，更多的是使用运算符，不在局限于基本门级元件。其下可以触及门级电路之间的连接关系，上可以扩展到电路的功能描述。

行为级建模

行为级建模，我认为就是用多个并行的，动态的过程模块去描述系统的工作。主要就是去描述电路所具有的行为，或者说，就是当电路在哪些输入信号来临时会有怎样的输出。

说明 符号

错误输出覆盖重定向 2>

实例：

同时重定向标准输出和错误输出：使用 &> 或 &>> 操作符将程序的标准输出和错误输出都发送到同一个文件。

  • 使用 2>&1 将标准错误输出重定向到标准错误输入：

    示例：command > output_and_errors.txt 2>&1

  • 使用 &>> 将输出追加到现有文件：

    示例：command &>> output.txt

垃圾桶黑洞装置/dev/null: 将不需要的输出重定向到/dev/null

命令执行条件:

  • ; : 直接连接两条指令，不考虑相关性

    示例：ls abc; touch tmp

  • 与 && : 前一个指令执行正确（返回值0），执行后一个指令

  • 或 || : 前一个指令执行错误（返回值非0），执行后一个指令

  • 管道 |： 将一个命令的标准输出作为下一个命令的标准输入。

    示例：cat file.txt | wc -l

  • 后台执行 & ：命令行末尾添加，将指令放到后台执行，不影响当前shell操作

​ 示例： wget http://example.com/large_file.zip &
